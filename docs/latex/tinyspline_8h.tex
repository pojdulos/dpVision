Buckles {\ttfamily spline} by {\ttfamily b} and stores the result in {\ttfamily \+\_\+buckled\+\_\+}. Creates a deep copy of {\ttfamily spline}, if {\ttfamily spline} != {\ttfamily \+\_\+buckled\+\_\+}.

This function is based on\+: \begin{DoxyVerb} Holten, Danny. "Hierarchical edge bundles: Visualization of adjacency
 relations in hierarchical data." Visualization and Computer Graphics,
 IEEE Transactions on 12.5 (2006): 741-748.
\end{DoxyVerb}
 Holten calls it "{}straightening"{} (page 744, equation 1).

Usually, the range of {\ttfamily b} is\+: 0.\+0 \texorpdfstring{$<$}{<}= {\ttfamily b} \texorpdfstring{$<$}{<}= 1.\+0 with 0 yielding to a line connecting the first and the last control point (no buckle) and 1 keeping the original shape (maximum buckle). If {\bfseries{\texorpdfstring{$<$}{<}}} 0 or {\bfseries{\texorpdfstring{$>$}{>}}} 1 the behaviour is undefined, though, it will not result in an error.

On error, (and if {\ttfamily spline} != {\ttfamily \+\_\+buckled\+\_\+}) all values of {\ttfamily \+\_\+buckled\+\_\+} are set to 0/\+NULL.

Computes the derivative of {\ttfamily spline}, see\+: \begin{DoxyVerb}http://www.cs.mtu.edu/~shene/COURSES/cs3621/NOTES/spline/B-spline/bspline-derv.html

 The derivative of a spline {\itshape s} of degree {\itshape d} with {\itshape m} control points and {\itshape n} knots is another spline {\itshape s\textquotesingle{}} of degree {\itshape d-\/1} with {\itshape m-\/1} control points and {\itshape n-\/2} knots, defined over {\itshape s} as\+:

\begin{eqnarray*}
  s\textnormal{\textquotesingle}(u) &=& \sum_{i=0}^{n-1} N_{i+1,p-1}(u) *
        (P_{i+1} - P_{i}) * p / (u_{i+p+1}-u_{i+1}) \\
        &=& \sum_{i=1}^{n} N_{i,p-1}(u) *
        (P_{i} - P_{i-1}) * p / (u_{i+p}-u_{i})
\end{eqnarray*}

If {\itshape s} has a clamped knot vector, it can be shown that\+:

\begin{eqnarray*}
  s\textnormal{\textquotesingle}(u) &=& \sum_{i=0}^{n-1} N_{i,p-1}(u) *
        (P_{i+1} - P_{i}) * p / (u_{i+p+1}-u_{i+1})
\end{eqnarray*}

where the multiplicity of the first and the last knot value {\itshape u} is {\itshape p} rather than {\itshape p+1}.

On error, (and if {\ttfamily spline} != {\ttfamily \+\_\+derivative\+\_\+}) all values of {\ttfamily \+\_\+derivative\+\_\+} are set to 0/\+NULL.
